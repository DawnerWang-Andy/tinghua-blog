## 主线与副线

### 主线：控制如何伪装成自主

发展路径：

收养的善意 -> 环境设计 -> 记录系统 -> 选择预筛选 -> 数据施压 -> 小野识别机制 -> 小野公开讲述机制 -> 小野发现自己也在使用机制。

### 陈远平线：从“我没强迫”到“我不知道”

关键变化：

- 早期：每个动作都有专业解释。
- 中期：笔记写不下去，物件开始失序。
- 后期：拆弹簧扣，转椅子，划掉姓，接受半开状态。

### 小野线：从“听话”到“保留问题”

关键变化：

- 童年：学会读懂大人话里的真正意思。
- 青少年：开始关门、质疑椅子和志愿表。
- 青年：通过他人案例识别自己的经验。
- 成人：写书、讲座、卖书，同时质疑自己的书是否也在制造控制。

### 继承线

陈建国的空间权威 -> 周秀兰的温柔控制 -> 陈远平的引导式控制 -> 林小野的分析式控制。

这条线已经很清楚。后续不要再解释更多，应该让物件继续证明。

## 最需要加强的因果断点

1. 小野从第 16 章发现旧教材，到第 18 章写书，中间可以再补一个“尝试不写但失败”的小过渡。
2. 第三部书突然畅销，可以增加一个社会传播节点：短视频切片、公众号书评、家长群传播、争议文章等。
3. 小野讲座从私人剖析到商业化产品，需要更多外部压力推动，不只靠他自省。
4. 陈远平退休、读书、转椅子的变化很细，但外部生活较单一。可增加一个邻居/书店店员/旧同事的轻触发，避免全部靠独处完成。

## 重复点

| 重复模式 | 出现位置 | 建议 |
|---|---|---|
| 陈远平观察并记录 | 2-7 多章 | 保留代表性场景，压缩相似解释 |

| 小野意识到自己像陈远平 | 14、16、18、23、30 | 保留递进，删掉同义反复 |
