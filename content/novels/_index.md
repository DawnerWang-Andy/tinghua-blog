## 索引

| | |
|---|---|
| **书名** | 听话 |
| **篇幅** | 五部 · 四十七章 · 约17万字 |
| **类型** | 长篇文学小说 |
| **主题** | 温柔的控制 · 代际继承 · 被安排的童年 |
| **一句话** | 陈远平从不命令，从不打骂，从不强迫。他只是记录。只是分析。只是在每一次"你觉得呢"之后，等着小野走进他准备好的选项里。 |
| **没有坏人，但也没有人是完全无辜的。** | |

### 人物

| | |
|---|---|
| **陈远平** | 教育研究者。理性、温和、从不发脾气。把观察当关心，把记录当爱。在心里给自己打勾。 |
| **林小野** | 八岁被领养，二十年后写了一本书分析养父。用的恰好是养父教他的方法。 |
| **张素兰** | 陈远平的母亲。小学语文教师。"跑什么跑。坐着。"把漫画换成单词书。把纠正当关心。 |
| **陈国栋** | 陈远平的父亲。工程师。同一把藤椅，同一份报纸。四十年，从来没有看过儿子一眼。 |

### 意象

| | |
|---|---|
| **门** | 弹簧吸扣吸在墙上——"通风好"。从开着的门到关上的门，到停在四十五度的门。 |
| **两把椅子** | 朝南有阳光，朝北对着墙。十五年后各自转了角度，对着同一块空地。 |
| **两杯茶** | 一杯给自己，一杯放在空椅子面前。倒了二十年——后来不倒了。 |
| **笔记本** | "观察记录——林小野"→划掉"林"→"小野"。二十年，二十一本。"第一次自主放弃——非强迫。" |
